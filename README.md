以下、コンテスト当日のプレゼンより抜粋
本システムは音楽プレイリストを用いた勉強支援webアプリケーションで、
｢音楽が鳴っている間は必ず勉強する｣というルールを提供し
任意の合計再生時間のプレイリストを再生し、毎日の勉強を支援します。
特徴であるkey_songは時間を直感的に認識させ勉強をせかす効果を持ちます。
短期集中型勉強法の理想的な形をとると私は考えます。
